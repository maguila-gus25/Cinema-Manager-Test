Cinema Manager

Um sistema completo de gerenciamento de cinema desenvolvido em Python

📋 Sobre o Projeto

O Cinema Manager é um sistema robusto para gerenciamento de cinemas, desenvolvido em Python com banco de dados SQLite. O sistema permite o controle completo de filmes, salas, sessões, ingressos e vendas, oferecendo uma solução completa para administradores de cinemas.

✨ Funcionalidades Principais

🎬 Gestão de Filmes

  • Cadastro de filmes com título, duração, gênero e classificação
  • Atualização e remoção de filmes
  • Controle de sinopse e informações detalhadas

🎪 Gestão de Salas

  • Cadastro de salas com número, tipo e capacidade
  • Gerenciamento de assentos por sala
  • Controle de disponibilidade de assentos

🎫 Gestão de Sessões

  • Agendamento de sessões por filme e sala
  • Controle de horários e datas
  • Monitoramento de ingressos disponíveis

💰 Gestão de Vendas

  • Registro de clientes
  • Processamento de vendas de ingressos
  • Geração de recibos
  • Histórico de compras

🛠️ Tecnologias Utilizadas

Python SQLite SQL POO

🚀 Como Começar

Para começar a usar o Cinema Manager, siga estes passos:

# Clone o repositório git clone https://github.com/seu-usuario/cinema-manager.git # Instale as dependências pip install -r requirements.txt # Execute o sistema python main.py

📚 Estrutura do Projeto

cinema-manager/ ├── database.py # Configuração do banco de dados ├── filme.py # Classe de gerenciamento de filmes ├── sala.py # Classe de gerenciamento de salas ├── sessao.py # Classe de gerenciamento de sessões ├── ingresso.py # Classe de gerenciamento de ingressos ├── cliente.py # Classe de gerenciamento de clientes ├── venda.py # Classe de gerenciamento de vendas └── main.py # Script principal de execução
